## 🎨 Customize in 5 Minutes

### Change Brand Color
Edit `tailwind.config.ts`:
```typescript
colors: {
  accent: '#your-color',
  accentAlt: '#your-color-2',
}
```

### Update Content
Edit `src/app/page.tsx` - Change any text directly

### Modify Animations
Edit component files - Change `duration`, `delay`, `ease`
